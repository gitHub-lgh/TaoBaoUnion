# TaoBaoUnion
淘宝领券App
App主要用途：领取淘宝商品的优惠卷
业务功能：
1、首页：搜索框。商品分类导航。轮播图。商品展示，包括商品的标题、图片、原价、券后价、以及购买人数。
2、精选页：页面左右两部分组成，左边商品的分类导航，右边对应商品展示
3、特惠页：商品列表
4、搜索页：页面由搜索框、搜索历史、热门搜索关键词组成。可在搜索框输入内容发起搜索，以列表的形式展示搜索结果。
